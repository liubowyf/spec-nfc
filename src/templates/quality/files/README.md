# `quality` 模块使用说明

这个目录存放测试、回归和发布验证文件。

## 文件

- `test-strategy.md`：测试分层与覆盖原则
- `regression-checklist.md`：通用回归清单
- `release-verification.md`：发布前后验证模板
- `test-case-template.md`：测试用例模板
- `bugfix-regression-template.md`：缺陷回归模板
- `AGENT.md`：给 AI 直接使用的质量规则

## 使用时机

- 新功能上线
- bugfix 修复
- 高风险改动发布前
- 需要把验证动作交给下游继续执行
