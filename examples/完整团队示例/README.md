# 完整团队示例

执行：

```bash
specnfc demo --cwd /tmp/specnfc-demo
```

重点看：

- `.specnfc/context/`
- `.specnfc/execution/`
- `.specnfc/governance/`
- `.specnfc/governance/personal-skills.md`
- `.specnfc/design/api/`
- `.specnfc/design/db/`
- `.specnfc/quality/`
- `specs/changes/risk-device-link/`

```bash
specnfc doctor --cwd /tmp/specnfc-demo
specnfc explain skills --cwd /tmp/specnfc-demo
specnfc change check risk-device-link --cwd /tmp/specnfc-demo
```
