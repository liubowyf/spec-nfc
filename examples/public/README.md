# 公开示例

本目录只保留适合公开仓说明的最小示例。

推荐按下面顺序阅读：

## 1. `minimal-init/`

最适合第一次接触 `specnfc` 的读者。

你会先看到：

- `.specnfc/`：控制面
- `.nfc/`：运行时与协作层
- `specs/`：正式 dossier
- `AGENTS.md / CLAUDE.md / .trae/rules/project_rules.md / opencode.json`：入口投影

## 2. `demo-output/`

适合理解 `specnfc demo` 生成的更完整 enterprise profile 公开结构。

推荐配合阅读：

- `specs/public-samples/change-full/`
- `specs/public-samples/integration-full/`

## 建议的整体阅读路径

1. 先读 `minimal-init/`
2. 再读 `specs/public-samples/change-full/`
3. 然后读 `specs/public-samples/integration-full/`
4. 最后再看 `demo-output/`

这样更容易建立从“初始化 → 变更推进 → 集成对齐 → 完整演示仓”的整体认知。
