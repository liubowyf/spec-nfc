# Capabilities

## 需增强的能力面
1. **协议接管能力**：把初始化从目录生成提升到项目协议接入。
2. **多层索引能力**：把项目记忆升级为 team/project/repo/change/integration 分层索引。
3. **阶段化流程能力**：统一 `clarify/design/plan/execute/verify/accept/archive` 阶段机。
4. **next-step protocol**：每个阶段输出固定的下一步建议，而不是只报错或只列目录。
5. **skill-pack 快照能力**：团队技能源 -> 项目快照 -> 仓内执行的稳定链路。
6. **入口投影能力**：让多工具入口文件严格从 `.specnfc/` 派生。
7. **渐进迁移能力**：旧仓不重置、不爆炸式迁移。
